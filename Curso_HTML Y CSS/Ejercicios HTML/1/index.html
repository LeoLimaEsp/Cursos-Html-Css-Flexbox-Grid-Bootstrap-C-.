Etiquetas de bloque VS etiquetas de linea

La etiqueta de bloque por excelencia es div, de bloque es aquella que genera saltos de linea con sus etiquetas hermanas, son las envidiosas
La etiqueta de bloque
La etiqueta de linea por excelencia es span, de linea solo ocupa el espacio necesario que tiene su contenido, no es envidiosa Hola, soy otra etiqueta span ☝🏻
